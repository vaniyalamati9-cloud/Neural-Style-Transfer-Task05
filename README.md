
Neural Style Transfer - Task 05

Project Overview
This project applies artistic style of famous paintings
to content images using Neural Style Transfer.
Built as part of AI Internship Task 05.

Technologies Used
- Python
- PyTorch
- VGG19 Pretrained Model
- Neural Style Transfer Algorithm
- Google Colab T4 GPU

 How It Works
1. Load content image and style image
2. Extract features using VGG19
3. Calculate gram matrices for style
4. Optimize target image to match both
5. Generate stylized output image

Results
- Successfully applied painting style to content image
- Model trained for 100 steps
- Combined content and style beautifully
